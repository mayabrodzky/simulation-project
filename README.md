# Polaris – Intelligent Simulation Platform

**Transforming decisions with intelligent simulations and turning complex data into interactive experiences.**

---

## 📖 About The Project

Polaris is an interactive simulation platform designed to empower managers and team leaders in making complex strategic decisions. The project was born out of the understanding that traditional tools like spreadsheets and static dashboards are insufficient for visualizing the dynamic impact of management choices.

The platform enables users to model intricate scenarios, test critical decisions in a risk-free environment, and communicate their potential impact with visual precision.

This repository contains two main components:
1.  **Marketing Website (`index.html`):** A professional landing page that introduces the product, its features, and the value it provides to clients.
2.  **Lab Simulation (`lab-simulation.html`):** A fully interactive product demo that showcases the core functionalities of the system in a professional lab setting.

## ✨ Key Features

* **Interactive Isometric Simulation:** A rich visual environment simulating the operations of a professional laboratory.
* **Resource Management:** Full control over budget, materials, and samples within the simulation.
* **Staff & Task Management:** Assign tasks to staff members with unique skills, manage shifts, and monitor their status in real-time.
* **Dynamic Scenarios:** Handle critical tasks and respond to dynamic emergency events that require quick and strategic decision-making.
* **Risk-Free Sandbox Mode:** A safe planning environment to test various strategies and action plans without affecting the live simulation.
* **Action Log & Session History:** Every action in Sandbox mode is logged for review. The entire scenario is saved in the "Sandbox History," allowing for detailed analysis and comparison of different strategies.
* **Modern User Interface:** A clean, intuitive, and responsive UI that makes the complex system easy to understand and operate.
* **No Installation Required:** The platform is fully browser-based, requiring no setup or installation.

## 🚀 Live Demo

## ⚙️ How To Run

This project is built with static HTML, CSS, and JavaScript and requires no special setup or backend server.

1.  Ensure all project files (`index.html`, `lab-simulation.html`, and any associated images) are located in the same directory.
2.  Open the `index.html` file in any modern web browser (e.g., Chrome, Firefox, Edge).
3.  You can navigate to the demo from the website or open the `lab-simulation.html` file directly.
